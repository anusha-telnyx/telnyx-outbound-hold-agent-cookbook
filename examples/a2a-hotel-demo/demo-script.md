# Demo Recording Checklist

Use this for a short screen recording of the working demo.

1. Show the local server running with `hold-agent serve`.
2. Show the outbound call command using the fake hotel number.
3. Open Telnyx Conversation Insights for the call.
4. Show the fake hotel greeting and reservations menu.
5. Show the `send_dtmf` tool call and audible DTMF selection.
6. Show the hold phrase and assistant stop behavior.
7. Show the representative assistant starting with prior context.
8. Show the hotel booking confirmation.
9. Show both sides saying goodbye once.
10. Show the `end_call` tool call ending the call.
